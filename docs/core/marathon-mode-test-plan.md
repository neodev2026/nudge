## TC-MM-07: TTS 자동 재생

**분류**: 기능
**사전 조건**: Marathon Mode 스트림 진행 중
**절차**:
1. 발음 듣기 버튼이 있는 카드(단어/예문) 진입 시 소리를 확인한다.
2. 설명 카드(발음 버튼 없음) 진입 시 소리를 확인한다.

**기대 결과**:
- 발음 버튼 있는 카드: TTS 2회 자동 재생된다.
- 발음 버튼 없는 카드: TTS 재생 없음.

## TC-MM-08: 미니 퀴즈 출제

**분류**: 기능
**사전 조건**: Marathon Mode 스트림 진행 중
**절차**:
1. 카드를 넘겨 5개 스테이지를 완료한다.
2. 미니 퀴즈 화면을 확인한다.

**기대 결과**:
- 5문항이 표시된다.
- 각 문항은 방금 본 5개 스테이지 중 하나를 대상으로 한다.
- `word_to_meaning`과 `meaning_to_word` 유형이 혼합되어 있다.
- 오답 보기 3개는 모두 같은 상품 내 다른 스테이지에서 추출된다.

## TC-MR-05: 학습 랭킹 — DM 링크 진입 후 반영

**분류**: 기능
**사전 조건**: 활성 시즌 존재, DM 링크로 resume URL 진입 가능
**절차**:

## TC-MR-06: 퀴즈 랭킹 — 완주 점수 반영

**분류**: 기능
**사전 조건**: account-A, 활성 시즌 존재, 전체 퀴즈 완주 1회
**절차**:
1. 활성 시즌 기간 중 Marathon Mode를 완주한다.
2. `/marathon-ranking` → `퀴즈 랭킹` 탭을 확인한다.

**기대 결과**:
- account-A가 랭킹에 표시된다.
- `best_score_pct`가 실제 퀴즈 정답률(score/total_questions × 100, 반올림)과 일치한다.
- `attempt_count`가 완주 횟수와 일치한다.
- 동점자 정렬: best_score_pct DESC → attempt_count ASC 순서로 표시된다.

## TC-MR-07: 시즌 전환 (current / last)

**분류**: 기능
**사전 조건**: 활성 시즌과 종료 시즌 모두 존재
**절차**:
1. `/marathon-ranking`에서 "지난 시즌" 버튼(또는 토글)을 클릭한다.
2. URL이 `?season=last`로 변경되는지 확인한다.
3. 다시 "현재 시즌" 버튼을 클릭한다.

**기대 결과**:
- `?season=last` 시 종료 시즌의 시즌 정보와 랭킹이 표시된다.
- `?season=current` 또는 파라미터 없을 시 현재 시즌 랭킹이 표시된다.
- 시즌 전환 시 페이지가 새로고침 없이 업데이트된다.
